多说 API 接口文档

获得用户数据

GET /users/profile.json

  • user_id* 用户ID

请求范例

$ curl -X GET \
  http://api.duoshuo.com/users/profile.json? \
  user_id=378333

返回参数

  • code <Int> 结果码。0 为成功。失败时为错误码。
  • errorMessage <String> 错误消息。当 code 不为 0 时，返回错误消息。
  • response <Object> 当 code 为 0 时，返回请求到的 JSON 对象。
