多说 API 接口文档

获得当前文章的评论接口

GET /threads/listPosts.json

  • short_name* <String> 站点注册的多说二级域名
  • thread_key* <String> 当前文章的 thread-key
  • page* <String>
  • limit* <Int>
  • order <String> desc|asc 返回的评论的排序规则

请求范例

$ curl -X GET \
  http://api.duoshuo.com/threads/listPosts.json? \
  order      = asc& \
  thread_key = article1& \
  short_name = official& \
  page       = 1& \
  limit      = 10
